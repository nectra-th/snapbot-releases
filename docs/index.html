Getting Started

Up and Running in Minutes

Follow these steps to set up your first automation.

1

Download & Launch

▾

Download SnapBot.exe and run it. Windows SmartScreen may show a warning for new apps — click "More info" then "Run anyway". The app requests admin privileges for window capture access.

2

Select Your Target Window

▾

Click "Select Window" in the toolbar and pick the application or game you want to automate. The preview panel shows a live feed of the target window.

3

Create Your First Template

▾

Click "Click" in the sidebar draw bar. Draw a rectangle on the preview for the search area, then draw a smaller rectangle inside for the template image. You can also just click inside the area to use it entirely as the template.

Tip: Make search areas as small as possible for faster matching. Templates should be unique, stable parts of the UI.
4

Adjust the Match Threshold

▾

Each template card shows a live match percentage bar. Drag the threshold slider to control sensitivity — higher values mean stricter matching (fewer false positives). Default is 65%.

5

Press Play

▾

Press F9 or click Play. SnapBot continuously scans and performs actions when templates match. Press F9 again to stop. Use "BG" mode to automate without moving your real mouse.

Tip: Use "BG" (background) mode to keep using your PC while automation runs. Switch to "Mouse" mode if background clicking doesn't work with your target app.
6

Explore Advanced Features

▾

Drag: Draw source + destination for drag-and-drop actions.
Wait: Click only after an image has been visible for N seconds.
HotKey: Fire keyboard sequences when an image matches. Type "/" for special keys.
Cond: If/else logic — check one image, click another if not found.
Multi: Match against multiple images simultaneously.

7

Tips & Tricks

▾

Reorder: Drag cards to change execution order (top = highest priority).
No Repeat: Fire a template only once per session.
Offset: Click the crosshair to pick an exact click point relative to the match.
Delay: Add before/after delays to pace automation.
Scan Interval: Lower = faster scanning but more CPU usage.

FAQ

Frequently Asked Questions

Is SnapBot safe to use?

▾

Yes. SnapBot uses pixel-level image recognition only — it never reads or injects into game memory. It's a single .exe with no installer, no registry modifications, and no background services. It's as safe as taking a screenshot.

Will I get banned from games?

▾

SnapBot operates at the pixel level and does not modify game files or memory. However, automating games may violate their Terms of Service. Use SnapBot responsibly and check each game's automation policy. We recommend using it for single-player games and desktop productivity tasks.

What does "lifetime updates" mean?

▾

Every future update to SnapBot is included at no additional cost. The app automatically checks for updates on startup. You always get the latest features, bug fixes, and improvements — forever.

What games and applications does it work with?

▾

SnapBot works with any application that renders to a window on Windows 10/11. This includes games (both windowed and borderless-windowed), browsers, desktop apps, and more. Some games with strict anti-cheat (kernel-level) may block window capture — use background mode as a fallback.

Does it work with any screen resolution?

▾

Yes. SnapBot captures the target window directly, not your monitor. Templates match based on the window content regardless of your screen resolution or DPI scaling.

Can I use it while doing other things?

▾

Absolutely. Background mode sends clicks and keystrokes directly to the target window without moving your real mouse. You can browse, chat, or work on other tasks while SnapBot automates in the background.
