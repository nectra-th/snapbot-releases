Features

🖱️

Click Automation

Draw a search area and an image template. When the image is found on screen, SnapBot clicks it automatically.

↔️

Drag & Drop

Match a source image and drag it to a destination. Perfect for inventory management and drag-based interfaces.

⏱️

Wait & Click

Wait for an image to appear for a specified duration before clicking. Great for loading screens and popups.

🔀

Conditional (If/Else)

Check if one image exists, and if not, click another. Build smart decision-making automation flows.

⌨️

HotKey Actions

When an image is found, fire a key sequence instead of clicking. Supports special keys with DOWN/UP/PRESS.

🎯

Multi-Template

Match multiple source images to find the best one, then drag or click. Handles dynamic interfaces.

👻

Macro Recorder

Record and play back sequences of actions. Chain multiple steps into reusable macros.

Profiles

Getting Started

1

Download & Launch

Download SnapBot.exe and run it. Windows may show a SmartScreen warning for unsigned apps — click "More info" then "Run anyway". The app will request admin privileges for window capture.

2

Select a Target Window

Click "Select Window" in the toolbar and pick the application you want to automate from the list. The preview panel will show a live feed of the target window.

3

Create a Click Template

Click the "Click" button in the sidebar. Draw a rectangle on the preview to define the search area — where SnapBot looks for your image. Then draw a smaller rectangle inside for the template image — the image SnapBot will match. You can also click inside the search area to use the entire area as the template.

Tip: Make the search area as small as possible for faster matching. The template should be a unique, stable part of the UI.
4

Adjust the Threshold

Each template card shows a match percentage bar. Drag the threshold slider to set how similar the on-screen image must be to your template. Higher = stricter matching (fewer false positives). Default is 65%.

5

Hit Play

Press F9 or click the Play button. SnapBot will continuously scan the target window and perform actions when templates match. Press F9 again to stop.

Tip: Use "BG" mode (background) to automate without moving your real mouse. Use "Mouse" mode if background clicking doesn't work with your target app.
6

Create a Drag Template

Click "Drag" to create a drag-and-drop action. First draw the source area and source image, then draw the destination area and destination image. When both match, SnapBot drags from source to destination.

7

Create a Wait Template

Click "Wait" to create a delayed click. Draw the area and image like a Click template. Set the wait duration — SnapBot will only click after the image has been visible for that long.

8

Create a HotKey Template

Click "HotKey" to fire keyboard input when an image matches. Draw area and image, then add key sequences in the card. Type text and press Enter for text input. Type "/" to open the special key picker (F1-F12, Shift, Ctrl, etc.). Right-click chips to set DOWN/UP/PRESS.

9

Create a Conditional Template

Click "Cond" for if/else logic. Draw a "check" area and image (the condition), then draw a "click" area and image (the action). If the check image is NOT found, SnapBot clicks the action image.

10

Multi-Template

Switch to the "Multi" tab for advanced multi-image matching. Click "Drag-Drop" or "Click". Draw source and destination areas, then add multiple images to each. SnapBot finds the best matching source and drags/clicks accordingly.

11

Tips & Tricks

Reorder: Drag cards to change execution order. Templates are checked top to bottom.
Enable/Disable: Toggle individual templates on/off with the eye icon.
Offset: Click the crosshair button to pick an exact click point offset from the template center.
No Repeat: Enable "no repeat" to fire a template only once per play session.
Delay: Add before/after delays to pace your automation.
Scan Interval: Adjust the toolbar slider to control how fast SnapBot scans (lower = faster but more CPU).
